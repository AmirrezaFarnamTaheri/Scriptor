# `scriptor-export-runner`

Native Pandoc export jobs with explicit argument vectors, temp workspaces, progress events, and cancellation.

## Shipped

- Pandoc discovery (`PATH` + `SCRIPTOR_PANDOC_PATH`).
- HTML, PDF, DOCX, LaTeX, ePub profiles with dry-run preview.
- Async job slot with stderr streaming and cancel.

See [`docs/release/PANDOC_STRATEGY.md`](../../docs/release/PANDOC_STRATEGY.md).
